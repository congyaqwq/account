# account 多人账单计算

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### 简介

多人共同消费时可以帮助计算各自的开销

### 应用场景

a,b,c,d 早上吃了生煎  
a,b,c 中午吃了火锅  
b,c,d 晚上吃了烧烤  
要计算各自的开销

### 说明

1. 名字不填或者相同时会去重
2. 填完第二部返回第一步删除值会影响第二部的结果
3. 代码优化中
